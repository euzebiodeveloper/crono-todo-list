# Crono Todo List (crono-todo-list)

Estrutura inicial para o app "Crono Todo List" (React + Node + MongoDB).

Pasta principal: `CronoTodoList`
- `backend/`: API com Express + Mongoose
- `frontend/`: App React (Vite)

Próximos passos:
- Instalar dependências em cada pasta: `npm install`.
- Criar `.env` em `backend` com `MONGO_URI`.
- Rodar backend: `npm run dev` (na pasta `backend`).
- Rodar frontend: `npm run dev` (na pasta `frontend`).
